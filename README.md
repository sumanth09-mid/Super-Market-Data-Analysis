### Super Market Performance Analyzer – Power BI Dashboard
## Project Overview
The Super Market Performance Analyzer is an interactive Power BI dashboard developed to analyze sales, profit, customer behavior, and regional performance using transactional retail data. The dashboard provides business insights through KPI metrics, category distribution, state-level analysis, and Year-over-Year trends.
##Objectives
• Analyze overall sales and profitability
• Identify top-performing states and product categories
• Compare monthly Sales and Profit Year-over-Year
• Evaluate customer segments and payment methods
• Support data-driven business decision-making
## Key Features
• KPI Cards: Total Sales, Total Profit, Quantity, Cities, Profit Margin
• Category, Segment, and Payment Mode distribution analysis
• Top 10 States by Sales and Profit
• Geographic performance analysis using Map visual
• Monthly Sales and Profit Year-over-Year comparison
## Technical Implementation
• Data Cleaning and Transformation using Power Query
• Data Modeling with proper relationships and Date Table
• DAX Measures for KPI calculations and YoY analysis
• Professional dashboard design with consistent color theme
## Tools & Technologies
• Microsoft Power BI
• DAX (Data Analysis Expressions)
• Power Query
• Superstore Retail Dataset
## Outcome
The dashboard transforms raw transactional data into an interactive, insight-driven reporting solution that enables business stakeholders to monitor performance, identify trends, and make strategic decisions.
